Text Area Character Counter

This project is a simple Text Area Character Counter built using HTML, CSS, and JavaScript.
It allows users to type a message in a text area while showing the number of characters typed and the remaining characters allowed.

Features

Real-time character counting

Displays typed characters and maximum limit

Shows remaining characters

Prevents typing beyond the character limit

Simple and clean layout

Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Project Structure
Text-area-count-visiblity
│
├── index.html
├── style.css
├── script.js
└── README.md
How It Works

The user types inside the text area.

JavaScript listens for the input event.

The script counts the number of characters typed.

The page updates the character counter in real time.

Example:

20 / 200 characters        180 remaining
How to Run the Project

Clone the repository

git clone https://github.com/daveena-navarathnam/Text-area-count-visiblity.git

Open the project folder.

Open index.html in a web browser.
